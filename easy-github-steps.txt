INITIALIZING REPO
1. git init

2. git add .

3. git commit -m 'first commit'

4. git status - to see where you are at'

5. git add .

6. git commit -m 'file created

MAKE A NEW BRANCH
1. git checkout -b feat/nameOfBranch
(SWITCH TO BRANCH FROM MAIN: git checkout feat/nameOfBranch)

2. git checkout

3. git checkout main

4. OPTIONAL git branch (lists all branches)

NOW WANT TO ADD FILES TO GITHUB
1. git add .

2. git commit -m 'changes made to file'

3. git push (should give error code)

4. git push --set-upsteam origin feat/nameOfBranch
(THIS PUSHES IT TO GITHUB FOR PR REVIEW)

MERGING BRANCHES INTO MAIN
1. git checkout main (switches back to main if PR is reviewed)

2. git pull (pull down the code from branch and merges into main)

3. Open Pull Requests in GitHub
